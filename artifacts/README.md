# 📂 Artifacts

This directory contains the intermediate artifacts generated during the execution of the Redrob AI Candidate Ranking pipeline.

These files are produced automatically at different stages of the pipeline and are reused by downstream modules to avoid redundant computation and improve reproducibility.

## Included Artifacts

- Candidate audit reports
- Engineered candidate feature tables
- Hybrid retrieval shortlists
- Cross-Encoder reranked candidates
- Final reasoning shortlist
- Pipeline metadata and summary reports
- Processed job description representation

Most of these artifacts are generated automatically during pipeline execution and can be regenerated by rerunning the corresponding pipeline stages.
